# ![dtmoney logo](https://github.com/hertaraujo/dtmoney/blob/main/src/assets/logo.svg)
A finance app to control your spending

### Try it here
[**dtmoney**](https://natours-hert.netlify.app/)

### Technologies
- TypeScript
- Reactjs / Next.js
- Styled Components / SCSS
- Axios
- Miragejs

### Run it locally:

    $ yarn
    $ yarn start
